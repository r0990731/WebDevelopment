Lossy compression

Dit wordt gebruikt om een bestand kleiner te maken qua geheugen. Maar het probleem hierbij is dat je de verloren data niet meer terug kan halen.

Lossless compression

Dit wordt ook gebruikt om het bestand kleiner te maken, maar het grote verschil is wel dat je de "verloren" data wel terug kan halen.

JPEG

JPEG gebruikt vooral lossy compression

PNG

PNG heeft een lossless compression

GIF

GIF gerbuikt ook een lossless compression

Als je een afbeelding herhaaldelijk bewerkt en bewaart in een lossy compression zal het kwaliteitsverlies van je afbeelding afnemen. Ook zal je minder details zien, maar de afbeelding zal minder data gebruiken.
